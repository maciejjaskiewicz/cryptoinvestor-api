## CryptoInvestor

**CryptoInvestor** is a system consisting of Web API and **[SPA website](https://github.com/maciejjaskiewicz/cryptoinvestor-spa)**, which allows to register account, create own cryptocurrency portfolio and manage it. System also show real-time coins price and history from external API.

**Demo** application you can find [here](http://demo.jaskiewicz.tech/cryptoinvestor).

## CryptoInvestor API
The **CryptoInvestor API** is a .Net Core application with JWT authentication and clean code architecture. This is a RESTful API that can be accessed by any type of application or tool that can handle the HTTP requests.
